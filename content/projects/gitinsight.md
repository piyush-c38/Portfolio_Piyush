GitInsight helps developers enter an unfamiliar repository and quickly answer the two questions that matter most: what does this codebase do, and where should I start?

The project combines AST-aware parsing, embeddings, and retrieval-augmented generation so the interface can summarize architecture, surface relevant files, and explain implementation paths without requiring manual spelunking.
